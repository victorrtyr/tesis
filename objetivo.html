4. Objetivo General y Objetivos Específicos

La formulación de objetivos establece con precisión las metas que se pretenden alcanzar. En el contexto de la presente investigación, los objetivos han sido diseñados alineados con las directrices nacionales de SUNEDU para trabajos de investigación aplicada en ciencias de la computación.

4.1. Objetivo General

Implementar un modelo de Random Forest entrenado con datos históricos de criminalidad y variables geoespaciales para determinar y visualizar en tiempo real las zonas de alto riesgo delictivo en la ciudad del Cusco durante el año 2025, con el propósito de apoyar la toma de decisiones preventivas de las autoridades.

4.2. Objetivos Específicos

  1. Recolectar y consolidar datasets históricos (2018-2025) de fuentes oficiales (INEI, PNP, Serenazgo).
  2. Diseñar pipeline de preprocesamiento y limpieza de datos con enriquecimiento geoespacial.
  3. Entrenar, validar y optimizar modelo Random Forest evaluando métricas de precisión, recall y F1-score.
  4. Desarrollar aplicación web "CuscoSeguro" (Node.js + React + Mapbox) para visualización de riesgos.
  5. Integrar modelo en backend mediante API REST para predicciones dinámicas.
  6. Evaluar efectividad mediante pruebas de usuario con autoridades locales.
