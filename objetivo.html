4. Objetivos de la Investigación

SMART Verified

4.1. Objetivo General

Implementar y evaluar un modelo Random Forest geoespacial para predecir zonas de alto riesgo delictivo en Cusco durante el periodo 2025, alcanzando un AUC-ROC ≥ 0.99 y un umbral óptimo de 0.607, integrándolo operativamente en la aplicación web "CuscoSeguro" para el apoyo preventivo de las autoridades.

(Tejada Castillo & Curo Mendoza, 2025)

4.2. Objetivos Específicos (Pipeline del Proyecto)

1

Datos Históricos

Recolectar y consolidar datasets criminales del periodo 2018-2025 (fuentes oficiales INEI, PNP DataCrim).

2

Preprocesamiento

Diseñar pipeline de limpieza y geocodificación para enriquecer la data cruda con coordenadas espaciales.

3

Entrenamiento RF

Entrenar y optimizar el modelo Random Forest buscando métricas superiores (Accuracy > 99%) y balance de clases.

4

Desarrollo Web

Desarrollar la aplicación full-stack "CuscoSeguro" integrando visualización de heatmaps interactivos (Mapbox).

5

Integración API

Conectar el modelo de ML con el frontend mediante una API REST/WebSockets para consultas en tiempo real.

6

Validación

Evaluar la efectividad y usabilidad de la herramienta con stakeholders locales y pruebas de usuario.
